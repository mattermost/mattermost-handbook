# Product Security

The Product Security team is responsible for identifying and managing solutions to protect Mattermost product security.

## Areas of Ownership
- Responsible Disclosure Program
- Security Reviews
  - Plugins
  - Features
  - Architecture
  - Release Pipelines
- Bug Bounty Program
- Threat Modeling
- Security Automation
  - Dependency Track
  - Security Scorecard
  - Gobom
- Penetration Testing (Product)
- Security Release Notes
